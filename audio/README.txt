DURARE - Cartella audio del player musicale

Mettici 10 file MP3 royalty-free nominati cosi:
  track-01.mp3 ... track-10.mp3

Dove trovarli:
  https://pixabay.com/it/music/search/allenamento/  (royalty-free, no attribuzione)
  https://mixkit.co/free-stock-music/
  https://www.bensound.com/

L app legge questi file con path relativi (./audio/track-XX.mp3).
Se i file mancano, il player mostra i nomi delle tracce ma non parte la musica.

Per personalizzare titoli/artisti nel player, apri il file DURARE_app.html
con un editor di testo e modifica la sezione: const TRACKS = [ ... ]
